## Manual (zh_cn)

### 运算符优先级

运算符 | 优先级 | 结合 | 备注
------|--------|------|----
+|1|左
-|1|左
*|2|左
/|2|左
%|3|左|取模运算
^|4|右
&|4|右|开方, a^b 等价于 a&(1/b)

功能：

- 计算功能
  - 可以计算绝大部分由双目运算符组成的表达式。
  - 支持多位数字的解析和负数的解析。
    - "-2+2", "2--2", "2-(-2)" 都是合法的表达式。
    - 因为不支持单目运算符，如果在非')'后出现负号则默认属于下一个数字。
    - "-2^2" 的意思是(-2)^2。

- 界面
  - 可以使用按钮输入也可以使用键盘输入。
  - 结果实时更新，等号实际上是个摆设。
  - 按C键可以清空。
